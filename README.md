# 🌍 Global Time

**Global Time** 是一个基于原生 HTML / CSS / JavaScript 构建的全球双时区时间显示应用，  
支持 **数字时钟 + 模拟时钟双显示**，并可自由选择任意时区进行对比查看。

项目注重：
- 🌐 全球化（多时区 & 中英文切换）
- 🕰️ 可读性（模拟 + 数字双时钟）
- 🎨 现代化 UI（深色渐变风格）
- ⚙️ 零依赖、即开即用

---

## ✨ 核心特性

- ⏱️ **双时区同时显示**
  - 主时区 & 第二时区可自由选择
- 🕰️ **模拟时钟 + 数字时钟**
  - 精确到秒，动态平滑更新
- 🌍 **基于 Intl API 的真实时区**
  - 自动处理夏令时（DST）
- 🌐 **中 / 英文界面切换**
- 🎨 **现代深色 UI 设计**
- 📦 **单文件部署**
  - 无需构建工具，无需后端
